Cómo hacer una calculadora básica con HTML, JavaScript y CSS

Hoy explicaré como crear un calculadora básica (suma, resta, división y multiplicacion) con HTML, JavaScript y CSS

No será una gran cosa, porsupuesto que se puede hacer de diferentes formas, pero apartir de esto podrás crearte la tuya más bonita, con más funciones y no solo con operaciones básicas.

Estructura en HTML

En este caso voy a usar

  • Tres areas de entrada de texto: etiqueta input, Dos para ingresar números y una para mostrar el resultado el cual será solo de lectura.
  • Cinco botones: etiqueta button, Cuatro para hacer las operaciones pero cada una de ellas con diferente contenido +, -, * y / y una para borrar (vaciar las entradas de texto y el resultado). Para más comodidad lo encapsulamos en un contenedor: etiqueta div.
  • Las rayas horizontales no son más que una raya con la etiqueta hr

Los botones

Vamos a usar el evento onclick mediante el cual cuando se clickee llame a la función operar('signo') mandando un signo por argumentos.
